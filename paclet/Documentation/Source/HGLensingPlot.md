## Usage

`HGLensingPlot[evolutionResult, stateId]` plots gravitational lensing (deflection angle vs impact parameter) for the state `stateId`, with the general-relativity prediction overlaid.

## Details

- Compares the discrete geometry's light bending to the GR expectation.
- See `Options[HGLensingPlot]` for the analysis and styling options.
