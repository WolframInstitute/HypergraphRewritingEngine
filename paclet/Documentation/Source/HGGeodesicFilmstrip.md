## Usage

`HGGeodesicFilmstrip[evolutionResult]` returns geodesic plots for each timestep.

## Details

- The per-timestep companion to `HGGeodesicPlot`.
- See `Options[HGGeodesicFilmstrip]` for the analysis and styling options.
